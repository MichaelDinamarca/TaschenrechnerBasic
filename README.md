# TaschenrechnerBasic
Test Projekt für Promotive GmbH

### Autor: *Michael Dinamarca*

## Anforderungen:
- `Java` Version 8 
- `Maven` Version 2

## Projekt bauen
```
mvn clean package
````

## Projekt ausführen
entweder
```
Java -jar target/taschenrechner.jar
```
oder die `taschenrechner.jar` Datei im `target` Ordner mit Doppelklick ausführen
